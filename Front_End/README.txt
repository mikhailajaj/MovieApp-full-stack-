# MyMoviesApp Frontend

This is the frontend part of MyMoviesApp, an iOS application built with Swift and SwiftUI. It follows the MVVM architectural pattern and connects to a RESTful backend for data management.

## Features

- Browse a list of movies
- Add new movies to the list
- Update and delete existing movies

## Setup

1. Clone the repository to your local machine.
2. Open the project in Xcode.
3. Configure the `NetworkService.swift` with the correct backend base URL.
4. Build and run the app on your chosen iOS simulator or device.

## Dependencies

- SwiftUI for the UI components.
- Combine for data binding and asynchronous programming.
